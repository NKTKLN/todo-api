# 📋 ToDo API

## 💾 Prerequisites

* [Golang](https://go.dev/)
* [Docker CE](https://docs.docker.com/engine/install/)
* [Docker Compose](https://docs.docker.com/compose/install/)

## ⚙️ Running api

* 🐳 Run in Docker

  ```
  make docker-build
  ```
* 📄 Build to binary file

  ```
  make build
  ```
* 💻 Local run

  ```
  make run
  ```
* 🧪 Test run

  ```
  make test
  ```
